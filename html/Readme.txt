📄 desktopPyLauncher HTML Viewer（簡易ビュアー）

これは JSON 形式で記述された デスクトップランチャー構成ファイルを、
ブラウザ上で視覚的に確認できる 簡易 HTML ビュアー です。

✅ 特徴

    表示専用（編集はできません）

    リンク／画像／テキスト（Markdown対応）で構成された プロジェクトビュー に対応

    .url や http:// で始まるリンクは ダブルクリックで別ウィンドウにオープン

    image タイプは embed（Base64埋め込み）でも表示可

    video タイプは未対応（枠だけ "Not Supported" 表示）


⚠️ 実行ファイルや .lnk の動作までは再現しません

    本ビュアーはあくまで 見た目確認ツール です。
    実際のショートカット動作やアプリ起動などには対応していません。


📂 ファイルの入れ替え方

    表示対象の JSON ファイルは data/default.json に置いてください。

    または、HTML 内の JavaScript 部分で fetch("data/default.json") を書き換えて、任意のパスに変更することも可能です。



🚫 ローカルファイルで直接開いても動きません

セキュリティ制限（CORS）により、
ローカルファイル（file:///～）で開くと JSON 読み込みに失敗します。


✅ 正しく動かすには HTTP サーバーが必要です

以下のように簡易サーバーを立ち上げて下さい：

cd このHTMLファイルのあるフォルダ
python -m http.server 7680

その後、ブラウザで以下のURLにアクセス：

http://localhost:7680/dpy_launcher_viewer_final.html

※ポート 7680 は例です。他と被る場合は変更して下さい。
（8080 はゲーム系ソフトやNodeアプリで競合しやすいため避けた方が無難です）


🔄 更新時の再読み込み

表示が更新されない場合は
Ctrl + F5（キャッシュ無視で強制リロード） を行ってください。


💡補足

将来的に .js ファイルに分離したり、const でJSONを直接埋め込む形式にすれば
サーバーなしのローカル起動にも対応可能です。
